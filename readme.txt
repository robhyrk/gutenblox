A simple Gutenberg block editor extension that adds customized blocks and features.

== Description ==
 
Add customized Gutenberg blocks for commonly used layouts and styles. This plugin uses WordPress'
amazing Gutenberg components to let users easily add blocks with layouts and features not readily 
available in the stock Gutenberg editor.

This plugin will continue to improve and add features based off of user feedback. Our aim is to create 
a simple, lean, and modern set of blocks that are responsive out of the box.

Currently this plugin contains a Two-Column Block to display a main image, title, description, 
and supporting image. Users have the ability to align content "left" or "right".
 
== Installation ==
 
1. Install plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add block in Gutenberg editor and select "Two-Column Block" under "Common".
4. Select "left" or "right" aligned under Settings > Block in the Gutenberg editor.
